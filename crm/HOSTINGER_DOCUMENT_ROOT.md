# Hostinger Document Root рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

## Рџа№ИЈ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: 404 Not Found

рЃЌрЃБ `https://onecar.ge/login` рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА, рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ document root рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў.

## ­ЪЊІ Laravel-рЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

Laravel-рЃерЃў `public` рЃцрЃЮрЃџрЃЊрЃћрЃарЃў **рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА document root**.

РћюРћђРћђ config/
РћюРћђРћђ database/
РћюРћђРћђ public/          Рєљ рЃћрЃА рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА document root
Рћѓ   РћюРћђРћђ index.php
Рћѓ   РћюРћђРћђ .htaccess

РћюРћђРћђ routes/

## ­ЪћД Hostinger-рЃќрЃћ Document Root-рЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 1: hPanel-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ (рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў)

1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ Hostinger-рЃўрЃА hPanel-рЃерЃў
2. Websites Рєњ Manage Рєњ Advanced Рєњ Document Root
3. рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ document root-рЃў:
   - **рЃФрЃЋрЃћрЃџрЃў:** `public_html`
   - **рЃљрЃ«рЃљрЃџрЃў:** `public_html/portal/public`

рЃљрЃю рЃЌрЃБ domain-рЃўрЃАрЃЌрЃЋрЃўрЃА:
   - **рЃФрЃЋрЃћрЃџрЃў:** `domains/onecar.ge/public_html`
   - **рЃљрЃ«рЃљрЃџрЃў:** `domains/onecar.ge/public_html/portal/public`

### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 2: .htaccess-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ (рЃЌрЃБ document root-рЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ рЃерЃћрЃБрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ)

рЃЌрЃБ document root-рЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ рЃерЃћрЃБрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ, рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `.htaccess` рЃцрЃљрЃўрЃџрЃў `public_html`-рЃерЃў:

**рЃцрЃљрЃўрЃџрЃў:** `~/domains/onecar.ge/public_html/.htaccess`

```apache
<IfModule mod_rewrite.c>
    RewriteEngine On
    RewriteRule ^(.*)$ portal/public/$1 [L]
</IfModule>
```

### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 3: Symbolic Link (рЃЌрЃБ SSH рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃњрЃљрЃЦрЃЋрЃЌ)

```bash

```

## РюЁ рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ

Document root-рЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ:

1. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ: `https://onecar.ge/`
   - рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА Laravel-рЃўрЃА welcome page рЃљрЃю login page

2. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ: `https://onecar.ge/login`
   - рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА login рЃњрЃЋрЃћрЃарЃЊрЃў

## ­ЪћЇ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃГрЃарЃљ

### "404 Not Found"
- рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ document root рЃљрЃарЃўрЃА `portal/public`
- рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ, рЃарЃЮрЃЏ `public/index.php` рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА
- рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ, рЃарЃЮрЃЏ `public/.htaccess` рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА

### "500 Internal Server Error"
- рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ `storage` рЃЊрЃљ `bootstrap/cache` permissions (755)
- рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ `.env` рЃцрЃљрЃўрЃџрЃў
- рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ `APP_KEY` рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЌрЃБ рЃљрЃарЃљ

### "Class not found" рЃљрЃю "Vendor autoload"
- рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ: `composer install` (рЃЌрЃБ SSH рЃњрЃљрЃЦрЃЋрЃЌ)
- рЃљрЃю GitHub Actions-рЃЏрЃљ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЊрЃљрЃўрЃбрЃљрЃюрЃЮрЃА `vendor` рЃцрЃЮрЃџрЃЊрЃћрЃарЃў

## ­ЪЊЮ рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ

GitHub Actions workflow-рЃерЃў `exclude`-рЃерЃў рЃљрЃа рЃљрЃарЃўрЃА `vendor`, рЃарЃљрЃф рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ `vendor` рЃцрЃЮрЃџрЃЊрЃћрЃарЃў рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЊрЃљрЃўрЃбрЃљрЃюрЃЮрЃА. рЃЏрЃљрЃњрЃарЃљрЃЏ `vendor` рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ рЃљрЃа рЃљрЃарЃўрЃА Git-рЃерЃў (рЃљрЃарЃўрЃА `.gitignore`-рЃерЃў).

**рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃљ:** SSH-рЃўрЃЌ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ `composer install` Hostinger-рЃќрЃћ:

```bash
cd ~/domains/onecar.ge/public_html/portal
composer install --no-dev --optimize-autoloader
```
