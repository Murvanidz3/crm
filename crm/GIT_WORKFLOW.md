# Git Workflow - рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЋрЃљрЃњрЃќрЃљрЃЋрЃюрЃЮрЃЌ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў GitHub-рЃќрЃћ

## Рџа№ИЈ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў

**Git рЃљрЃа рЃљрЃарЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў!** рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃљрЃЊ рЃарЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃќрЃћрЃф рЃљрЃа рЃБрЃюрЃЊрЃљ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃЮрЃЌ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў, рЃўрЃАрЃўрЃюрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ **рЃљрЃа рЃљрЃўрЃАрЃљрЃ«рЃћрЃЉрЃљ** GitHub-рЃќрЃћ.

## ­ЪЊІ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў Git Workflow

### 1. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃљ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў рЃњрЃљрЃЦрЃЋрЃЌ

```bash
cd D:\crm
git status
```

рЃћрЃА рЃњрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃЌ:
- рЃарЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ (modified)
- рЃарЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃљрЃ«рЃљрЃџрЃўрЃљ (untracked)
- рЃарЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃЊрЃљрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ commit-рЃўрЃАрЃЌрЃЋрЃўрЃА (staged)

### 2. рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў Git-рЃерЃў

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў A: рЃДрЃЋрЃћрЃџрЃљ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ**
```bash
git add .
```

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў B: рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃў**
```bash
git add crm/app/Http/Controllers/CarController.php
git add crm/resources/views/cars/index.blade.php
```

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў C: CRM рЃцрЃЮрЃџрЃЊрЃћрЃарЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ**
```bash
git add crm/
```

### 3. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ Commit

```bash
git commit -m "рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ CarController рЃЊрЃљ cars view"
```

**рЃЎрЃљрЃарЃњрЃў commit messages:**
- `"Add new car status feature"`
- `"Fix login bug"`
- `"Update database migrations"`
- `"Add user management functionality"`

**рЃфрЃБрЃЊрЃў commit messages:**
- `"changes"`
- `"update"`
- `"fix"`

### 4. рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃћрЃЌ GitHub-рЃќрЃћ

```bash
git push origin main
```

## ­Ъћё рЃАрЃарЃБрЃџрЃў рЃърЃарЃЮрЃфрЃћрЃАрЃў (рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў)

```bash
# 1. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃАрЃбрЃљрЃбрЃБрЃАрЃў
cd D:\crm
git status

# 2. рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў
git add crm/

# 3. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ commit
git commit -m "Add new feature: car status tracking"

# 4. рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃћрЃЌ GitHub-рЃќрЃћ
git push origin main
```

## ­Ъџђ GitHub Actions Deploy

рЃарЃЮрЃфрЃљ `git push origin main` рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ:
1. РюЁ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў рЃњрЃљрЃЊрЃљрЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ GitHub-рЃќрЃћ
2. РюЁ GitHub Actions workflow рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃћрЃерЃЋрЃћрЃЉрЃљ (рЃЌрЃБ `crm/**` рЃцрЃљрЃўрЃџрЃћрЃЉрЃерЃў рЃўрЃДрЃЮ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ)
3. РюЁ Workflow рЃњрЃљрЃЊрЃљрЃўрЃбрЃљрЃюрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃА Hostinger-рЃќрЃћ FTP-рЃўрЃЌ

## ­ЪЊЮ рЃАрЃгрЃарЃљрЃцрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў

```bash
# рЃДрЃЋрЃћрЃџрЃљрЃцрЃарЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ рЃћрЃарЃЌ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃерЃў
cd D:\crm
git add . && git commit -m "Your message" && git push origin main
```

## Рџа№ИЈ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃў

### "Your branch is behind 'origin/main'"

```bash
git pull origin main
# рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЎрЃЋрЃџрЃљрЃЋ
git push origin main
```

### "Changes not staged for commit"

рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃа рЃЊрЃљрЃћрЃЏрЃљрЃбрЃљ Git-рЃерЃў. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ:
```bash
git add .
```

### "Nothing to commit"

рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ рЃБрЃЎрЃЋрЃћ committed-рЃўрЃљ. рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃљрЃњрЃќрЃљрЃЋрЃюрЃЮрЃЌ:
```bash
git push origin main
```

## ­ЪњА рЃарЃЕрЃћрЃЋрЃћрЃЉрЃў

1. **рЃарЃћрЃњрЃБрЃџрЃљрЃарЃБрЃџрЃљрЃЊ commit-рЃў:** рЃљрЃа рЃЊрЃљрЃћрЃџрЃЮрЃЊрЃЮрЃЌ рЃЊрЃўрЃЊ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃА
2. **рЃЎрЃљрЃарЃњрЃў commit messages:** рЃљрЃдрЃгрЃћрЃарЃћрЃЌ рЃарЃљ рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ
3. **рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ git status:** commit-рЃўрЃА рЃгрЃўрЃю рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃљ рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ
4. **рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ branches:** рЃЊрЃўрЃЊрЃў features-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў branch
