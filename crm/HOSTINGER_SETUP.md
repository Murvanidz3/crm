# Hostinger-рЃќрЃћ CRM-рЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ - рЃАрЃарЃБрЃџрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ

## РюЁ рЃарЃљ рЃњрЃљрЃЎрЃћрЃЌрЃЊрЃљ рЃБрЃЎрЃЋрЃћ:

1. РюЁ SQL рЃЉрЃљрЃќрЃљ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ: `crm_db`
2. РюЁ Database user рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ: `crm_db` / `Tormeti21!`
3. РюЁ SQL dump рЃўрЃЏрЃърЃЮрЃарЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ (рЃДрЃЋрЃћрЃџрЃљ рЃфрЃ«рЃарЃўрЃџрЃў рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ)
4. РюЁ Admin user рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ: `admin` / `admin123`

---

## ­ЪЊІ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў:

### 1. .env рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

Hostinger-рЃќрЃћ `portal` рЃцрЃЮрЃџрЃЊрЃћрЃарЃерЃў рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `.env` рЃцрЃљрЃўрЃџрЃў:

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў A: File Manager-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ**
1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ hPanel Рєњ File Manager
2. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ `portal` рЃцрЃЮрЃџрЃЊрЃћрЃарЃерЃў
3. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃцрЃљрЃўрЃџрЃў `.env`
4. рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЌ рЃЊрЃљ рЃЕрЃљрЃАрЃЋрЃўрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃў:

```env
APP_NAME=CRM
APP_ENV=production
APP_KEY=
APP_DEBUG=false

```

**Рџа№ИЈ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў:**
- рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ `APP_URL` рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЊрЃЮрЃЏрЃћрЃўрЃюрЃўрЃЌ (рЃЏрЃљрЃњ: `https://yourdomain.com`)
- `APP_KEY` рЃЊрЃљрЃбрЃЮрЃЋрЃћрЃЌ рЃфрЃљрЃарЃўрЃћрЃџрЃў - рЃерЃћрЃЏрЃЊрЃћрЃњ рЃюрЃљрЃЉрЃўрЃ»рЃерЃў рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃљ рЃЏрЃЮрЃ«рЃЊрЃћрЃЉрЃљ

---

### 2. APP_KEY-рЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў A: SSH рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃЌ (рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў)**

рЃЌрЃБ рЃњрЃљрЃЦрЃЋрЃЌ SSH рЃгрЃЋрЃЊрЃЮрЃЏрЃљ Hostinger-рЃќрЃћ:

```bash
cd ~/portal
php artisan key:generate
```

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў B: SSH рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ**

рЃЌрЃБ SSH рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃљрЃа рЃњрЃљрЃЦрЃЋрЃЌ, рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃ«рЃћрЃџрЃўрЃЌ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃЮрЃЌ APP_KEY:

1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ: https://generate-random.org/api-key-generator?count=1&length=32&type=mixed-numbers-symbols&prefix=
2. рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЌ рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў key
3. `.env` рЃцрЃљрЃўрЃџрЃерЃў `APP_KEY=` рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЕрЃљрЃАрЃЋрЃўрЃЌ: `APP_KEY=base64:YOUR_GENERATED_KEY_HERE`

рЃљрЃю рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃћрЃА рЃЮрЃюрЃџрЃљрЃўрЃю рЃбрЃБрЃџрЃў: https://laravel-key-generator.com/

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў C: PHP Script-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ**

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `generate_key.php` рЃцрЃљрЃўрЃџрЃў `portal` рЃцрЃЮрЃџрЃЊрЃћрЃарЃерЃў:

```php
<?php
echo 'APP_KEY=base64:' . base64_encode(random_bytes(32)) . PHP_EOL;
```

рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ (рЃЌрЃБ SSH рЃњрЃљрЃЦрЃЋрЃЌ):
```bash
php generate_key.php
```

рЃЊрЃљ рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЌ рЃерЃћрЃЊрЃћрЃњрЃў `.env` рЃцрЃљрЃўрЃџрЃерЃў.

---

### 3. Permissions-рЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

рЃЌрЃБ SSH рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃњрЃљрЃЦрЃЋрЃЌ, рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ:

```bash
cd ~/portal
chmod -R 755 storage
chmod -R 755 bootstrap/cache
```

рЃЌрЃБ SSH рЃљрЃа рЃњрЃљрЃЦрЃЋрЃЌ, File Manager-рЃерЃў:
- `storage` рЃцрЃЮрЃџрЃЊрЃћрЃарЃА рЃЊрЃљ `bootstrap/cache` рЃцрЃЮрЃџрЃЊрЃћрЃарЃА рЃЏрЃўрЃљрЃюрЃўрЃГрЃћрЃЌ 755 permissions

---

### 4. рЃерЃћрЃАрЃЋрЃџрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў

1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЊрЃЮрЃЏрЃћрЃўрЃюрЃќрЃћ: `https://your-domain.com/login`
2. рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ:
   - **Username:** `admin`
   - **Password:** `admin123`
3. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "рЃерЃћрЃАрЃЋрЃџрЃљ"

**Рџа№ИЈ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў:** рЃърЃўрЃарЃЋрЃћрЃџ рЃерЃћрЃАрЃЋрЃџрЃљрЃќрЃћ рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ рЃърЃљрЃарЃЮрЃџрЃў!

---

### 5. рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ

рЃерЃћрЃАрЃЋрЃџрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ:
1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃърЃарЃЮрЃцрЃўрЃџрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
2. рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ "рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ"
3. рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃърЃљрЃарЃЮрЃџрЃў

## ­ЪћД рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃГрЃарЃљ

### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: "No application encryption key has been specified"
**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ:** рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ `.env` рЃцрЃљрЃўрЃџрЃерЃў `APP_KEY` рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ. рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ `php artisan key:generate`.

### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: "SQLSTATE[HY000] [2002] Connection refused"
**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ:** рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ `.env` рЃцрЃљрЃўрЃџрЃерЃў database credentials:
- `DB_HOST=localhost` (рЃљрЃю Hostinger-рЃўрЃА рЃЏрЃўрЃћрЃа рЃЏрЃЮрЃгрЃЮрЃЊрЃћрЃЉрЃБрЃџрЃў host)
- `DB_DATABASE=crm_db`
- `DB_USERNAME=crm_db`
- `DB_PASSWORD=Tormeti21!`

### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: "The stream or file could not be opened"
**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ:** рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ `storage` рЃЊрЃљ `bootstrap/cache` рЃцрЃЮрЃџрЃЊрЃћрЃарЃћрЃЉрЃўрЃА permissions (755).

### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: "404 Not Found"
**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ:** 
- рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ `public` рЃцрЃЮрЃџрЃЊрЃћрЃарЃў рЃљрЃарЃўрЃА document root
- рЃљрЃю рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `.htaccess` рЃцрЃљрЃўрЃџрЃў `public` рЃцрЃЮрЃџрЃЊрЃћрЃарЃерЃў (рЃЌрЃБ рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА)

## РюЁ рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ

- [ ] `.env` рЃцрЃљрЃўрЃџрЃў рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃЊрЃљ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ
- [ ] `APP_KEY` рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ
- [ ] Database credentials рЃАрЃгрЃЮрЃарЃўрЃљ
- [ ] `storage` рЃЊрЃљ `bootstrap/cache` permissions рЃАрЃгрЃЮрЃарЃўрЃљ
- [ ] рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃерЃћрЃ«рЃЋрЃўрЃЊрЃћрЃЌ `/login` рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
- [ ] рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃерЃћрЃ«рЃЋрЃўрЃЊрЃћрЃЌ admin user-рЃўрЃЌ
- [ ] рЃърЃљрЃарЃЮрЃџрЃў рЃерЃћрЃфрЃЋрЃџрЃўрЃџрЃўрЃљ

## ­ЪјЅ рЃЏрЃќрЃљрЃЊрЃљрЃљ!

рЃЌрЃБ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃАрЃгрЃЮрЃарЃљрЃЊ рЃњрЃљрЃЎрЃћрЃЌрЃЊрЃљ, рЃљрЃ«рЃџрЃљ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ:

- рЃЊрЃљрЃљрЃЏрЃљрЃбрЃЮрЃЌ рЃЏрЃљрЃюрЃЦрЃљрЃюрЃћрЃЉрЃў
- рЃЏрЃљрЃарЃЌрЃЮрЃЌ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃў
- рЃюрЃљрЃ«рЃЮрЃЌ рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃћрЃЉрЃў
- рЃЊрЃљ рЃљ.рЃе.

рЃњрЃљрЃЦрЃЋрЃЌ рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃў? рЃЊрЃљрЃњрЃЋрЃўрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃўрЃЌ!
